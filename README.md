# Vn Native Framework Version 3 - Beta
Just beta version, so please don't use it for product at this version.
# How to install ? 
Very easy, first you need 
    `git clone https://github.com/steveleetn91/vn-native-3.git`
.Then run 
    `npm install`
# First run ?
Only run 
    `npm run build`
# How to run ?
Only run 
    `npm run serve`    
Then you can visit : 
    `http://localhost:9000`
Or
    `http://your_ip:9000`
